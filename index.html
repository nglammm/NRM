About NRM

NRM (Normal) is a programming language designed to simplify coding with easy-to-remember commands and a streamlined syntax. Whether you're new to programming or a seasoned developer, NRM offers a smooth learning curve and powerful functionality.

Key Features

  • Simple, intuitive syntax
  • Support for multiple data types
  • Functions for handling complex tasks easily
  • Cross-platform compatibility
  • Lightweight and fast

Basic Syntax

is x 10
